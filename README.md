# 🧮 Streamlit Simple Calculator
<img width="883" height="454" alt="image" src="https://github.com/user-attachments/assets/43ba1ead-78de-463e-9f6c-df3e65ec4e2d" />


A clean, interactive web-based calculator built using Python and the Streamlit framework. This project demonstrates basic arithmetic operations (Addition, Subtraction, Multiplication, Division) through an intuitive user interface.

## ✨ Features
* **Interactive UI:** Clean input fields for numbers and a dropdown for operations.
* **Real-time execution:** Instant calculation upon clicking the "Calculate" button.
* **Error Handling:** Built-in safeguards against logical errors, such as dividing by zero.
